th:ifAbove button

If top is true this should be visible on top of the page

If middle is true this should be visible between top and bottom

If bottom is true this should be visible at the bottom

th:ifBelow button

If top is true this should be visible on top of the page, but below the button

If middle is true this should be visible between top and bottom, but below the button

If bottom is true this should be visible at the bottom, but below the button
