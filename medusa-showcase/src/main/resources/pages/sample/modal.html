Custom modal CSS

By default, Medusa comes with a simple pre-defined style. If you want to customize the CSS, you can simply define custom rules

The following classes are used:

  • modal-background: Encompasses everything, contains rules on the darkness of the background
  • modal-container: Encompasses just the modal itself. Can be used for sizing, padding or overall rules
  • modal-header: Encompasses all header components (such as the title and the close button)
  • modal-title: h3 tag with title
  • modal-close: button tag that closes the modal
  • modal-content: div tag that wraps the entire content you provide in the m:modal tag
